# 🔐 Random Password Generator

A simple yet effective Python script that generates secure random passwords based on user-specified length.



## ✨ Features
- Generates random passwords with customizable length
- Includes all character types:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Special characters (!"#$%&'()*+,-./:;<=>?@[\]^_`{|}~)
- Simple and easy to understand code
- No external dependencies required

## 🎯 How It Works

The password generator follows these steps:

1. **User Input**: Asks the user for desired password length
2. **Character Pool**: Creates a string containing all possible characters
3. **Random Selection**: Randomly picks characters from the pool
4. **Password Construction**: Builds the password by repeating step 3
5. **Output**: Displays the generated password
